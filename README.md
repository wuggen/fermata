# Fermata, a MIDI data and music theory library

A personal project, for edification and entertainment.

Project goals:

- Parse and serialize the core MIDI spec, including translating to and from raw
  binary MIDI streams.
- Represent common music theory and performance constructs (scales, chords,
  dynamics, etc.) programatically.
- Enable relatively rapid progression from musical idea to software
  implementation; minimize fuss in the translation of theory to protocol.
- Force myself to learn more about music.
